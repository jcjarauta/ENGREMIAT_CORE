# SYSTEM OVERVIEW

ENGREMIAT se desarrolla como sistema local-first, modular y documentado.

## Componentes actuales
- Open WebUI: interfaz humana y Knowledge Base.
- Ollama: inferencia local.
- qwen3:14b: modelo local principal.
- PostgreSQL: persistencia backend.
- NestJS backend: base tecnica inicial.
- Knowledge Base: memoria documental consultable.

## Principio
Primero memoria estructurada; despues skills; despues agentes.
