# GUIA MANUAL - ENGREMIAT_CORE_FAST

## Objetivo
Crear una Knowledge Base ligera en Open WebUI para reducir latencia.

## Nombre recomendado
ENGREMIAT_CORE_FAST

## Archivo unico a subir
C:\ENGREMIAT_CORE\openwebui-knowledge\ENGREMIAT_CORE_FAST\ENGREMIAT_CORE_FAST_CONTEXT.md

## Pasos
1. Abrir Open WebUI en http://localhost:3000.
2. Ir a Espacio de Trabajo > Conocimiento.
3. Crear Conocimiento llamado ENGREMIAT_CORE_FAST.
4. Subir solo ENGREMIAT_CORE_FAST_CONTEXT.md.
5. Abrir chat con qwen3:14b.
6. Activar ENGREMIAT_CORE_FAST.
7. Repetir las 5 preguntas de validacion.
8. Comparar velocidad frente a ENGREMIAT_CORE.
