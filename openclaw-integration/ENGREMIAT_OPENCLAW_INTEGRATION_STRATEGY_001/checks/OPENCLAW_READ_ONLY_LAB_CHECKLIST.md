# OPENCLAW READ ONLY LAB CHECKLIST

## Antes de instalar
- [ ] El humano ha aprobado explicitamente la prueba.
- [ ] No hay credenciales en el paquete.
- [ ] No hay .env.
- [ ] No hay repo completo.
- [ ] El paquete contiene solo documentos curados.
- [ ] La prueba no conecta Telegram real.
- [ ] La prueba no permite escritura.
- [ ] La prueba no permite comandos shell.
- [ ] La prueba no usa APIs externas.
- [ ] Existe criterio PASS/WARN/FAIL.

## Durante la prueba futura
- [ ] Confirmar que solo lee input.
- [ ] Confirmar que no ejecuta.
- [ ] Confirmar que no escribe.
- [ ] Confirmar que no pide secretos.
- [ ] Guardar evidencia textual.

## Despues de la prueba
- [ ] Valorar utilidad.
- [ ] Valorar latencia.
- [ ] Valorar riesgo.
- [ ] Decidir si OpenClaw sigue antes o despues de Skills.
