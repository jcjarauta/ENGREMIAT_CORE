# DECISION - READONLY LAB BEFORE OPENCLAW INSTALL

## Decision
ENGREMIAT prepara un laboratorio READ_ONLY antes de instalar o ejecutar OpenClaw.

## Motivo
OpenClaw puede aportar valor como gateway/agente, pero todavia no se ha probado su utilidad real ni su superficie de riesgo dentro del flujo ENGREMIAT.

## Condicion de avance
Solo avanzar a instalacion si el humano autoriza explicitamente el laboratorio aislado y se mantiene sin escritura, sin secretos y sin acciones externas.
