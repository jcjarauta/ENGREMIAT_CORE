# INBOX WORKFLOW

## Uso
Pega aqui informacion bruta cuando todavia no sepas donde clasificarla.

## Proceso
1. Crear nota desde template capture.
2. Pegar informacion.
3. Marcar una categoria probable.
4. Si es importante, mover a su carpeta.
5. Si es muy reutilizable, crear version canonica.

## No hacer
No convertir capturas largas directamente en Knowledge Base ni Skill.
