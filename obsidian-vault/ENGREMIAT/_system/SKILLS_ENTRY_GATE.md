# SKILLS ENTRY GATE

## Objetivo
Definir cuando ENGREMIAT puede iniciar Skills sin automatizar prematuramente.

## Gate minimo
Para iniciar ENGREMIAT_OPENWEBUI_SKILLS_001 deben existir al menos 3 candidatos reales en skills-candidates.

## Cada skill-candidate debe incluir
- Entrada clara.
- Proceso repetible.
- Salida concreta.
- Reglas y limites.
- Evidencia verificable.
- Riesgo LOW/MEDIUM/HIGH.
- Decision humana: mantener como manual, convertir en Skill, aplazar o descartar.

## Skills iniciales recomendables
1. Resumir estado ENGREMIAT.
2. Leer roadmap y proponer siguiente paso.
3. Validar salida de bloque.
4. Crear checklist de cierre.
5. Preparar documento canonico para Open WebUI.

## No iniciar
No crear Skills si solo hay ideas sueltas, capturas brutas o procedimientos no validados manualmente.
