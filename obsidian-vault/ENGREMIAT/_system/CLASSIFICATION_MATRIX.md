# CLASSIFICATION MATRIX

| Pregunta | Si la respuesta es SI | Carpeta destino |
|---|---|---|
| Describe el estado actual o readiness? | Es estado operativo | status |
| Define etapas, bloques o pasos futuros? | Es roadmap | roadmaps |
| Registra una decision tomada? | Es decision | decisions |
| Explica estructura tecnica o componentes? | Es arquitectura | architecture |
| Enseña como hacer algo manualmente? | Es manual | manuals |
| Sirve para continuar una conversacion o instruir a un agente? | Es prompt | prompts |
| Define reglas, limites o politicas? | Es contrato | contracts |
| Es evidencia de una ejecucion o cierre? | Es report | reports |
| Es un procedimiento repetible con entrada/salida? | Es candidato a Skill | skills-candidates |
| Mejora la experiencia por menus, botones o mensajes? | Es UX Telegram | telegram-ux |
| Debe consultarse por RAG? | Preparar documento canonico | openwebui |

## Criterio de exportacion a Open WebUI
Exportar solo si el documento es corto, claro, actualizado y responde a una pregunta frecuente.

## Criterio de Skill
Solo candidato si tiene entrada, proceso, salida, reglas, evidencia y riesgo controlado.
