# MANUAL_GLOBAL_ARCHITECTURE

ENGREMIAT se organiza por entidades comunes: CORE, VAULT, PROJECT, CARD, SKILL, BUNDLE, ADAPTER, NODE, GATE y EVIDENCE.

Uso rapido:
1. Consultar HOME_GLOBAL.
2. Revisar REGISTRY_BOVEDAS y REGISTRY_PROYECTOS.
3. Trabajar en boveda dedicada.
4. Convertir trabajo en tarjetas.
5. Crear bundle si se usara OpenWebUI/Ollama.
6. Registrar evidencia y cierre.
