# KANBAN VIEW CONTRACT

## Objetivo

Definir como se generaran vistas Kanban desde objetos Obsidian.

## Columnas iniciales

- READY
- BLOCKED
- LEARNING
- IN_PROGRESS
- DONE

## Cada tarjeta debe mostrar

- id
- title
- type
- scope
- priority
- risk
- depends_on
- next_action

## Fuente

Los archivos de `03_OBJECTS`, no el tablero externo.
