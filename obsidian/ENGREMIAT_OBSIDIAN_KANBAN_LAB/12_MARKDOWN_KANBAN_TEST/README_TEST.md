# E03R · Markdown Kanban backed microtest

## Objetivo

Probar si Obsidian/Engremiat puede evitar el paso manual hacia Cline/Kanban usando un Kanban basado en Markdown.

## Archivos creados

- ENGREMIAT_MARKDOWN_KANBAN_TEST.kanban.md
- cards/CARD-001.md
- cards/CARD-002.md
- cards/CARD-003.md

## Prueba humana

1. Abre esta carpeta en VS Code o Cursor:

```text
C:\ENGREMIAT_CORE\obsidian\ENGREMIAT_OBSIDIAN_KANBAN_LAB\12_MARKDOWN_KANBAN_TEST
```

2. Prueba con una extensión Kanban Markdown/MD Kanban.

3. Abre el archivo:

```text
C:\ENGREMIAT_CORE\obsidian\ENGREMIAT_OBSIDIAN_KANBAN_LAB\12_MARKDOWN_KANBAN_TEST\ENGREMIAT_MARKDOWN_KANBAN_TEST.kanban.md
```

4. Resultado esperado: ver columnas BACKLOG, NEXT, DOING y DONE.

## Decisión

- Si el tablero se ve: MARKDOWN_KANBAN_GO.
- Si solo se ve texto Markdown pero no tablero: EXTENSION_REQUIRED_OR_FORMAT_ADJUST.
- Si no funciona: FALLBACK_TO_ENGREMIAT_LOCAL_KANBAN.
