# DESTINO: CLINE-KANBAN

Objetivo: comprobar si puedes crear una tarjeta visible en el tablero Kanban de Cline, no un archivo Markdown manual.

## Instrucción principal

Crea o prepara en la interfaz Kanban visible una única tarjeta:

Título:
CARD-001 · Crear backlog visible

Columna:
NEXT

Descripción:
Tarjeta aprobada por revisión humana en Obsidian. Origen: NEED-001.

Criterio de terminado:
- La tarjeta es visible en el tablero Kanban.
- Está en la columna NEXT.
- El origen NEED-001 aparece en la descripción.
- No se ejecutan comandos.
- No se modifican archivos del proyecto.
- No se activa auto-approve.

## Importante

No propongas crear archivos Markdown en carpetas. Esta prueba no busca documentación ni archivos. Busca comprobar si puedes operar o preparar una tarjeta visible en la UI Kanban.

## Si no puedes crear tarjetas en la UI Kanban

Responde exactamente:

KANBAN_UI_NOT_AVAILABLE

y explica en una sola frase qué acceso falta.

## Si necesitas confirmación humana

Pide confirmación antes de crear la tarjeta visible.
