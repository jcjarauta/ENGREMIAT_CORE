# OPENWEBUI FACTORY POLICY

## Permitido ahora
- Crear plantillas.
- Crear manifiestos.
- Crear exports locales.
- Preparar import asistido.
- Preparar API dry-run.

## No permitido ahora
- Escribir via API en Open WebUI.
- Crear Tools/Functions.
- Ejecutar codigo desde Skills.
- Dar permisos externos.
- Sincronizar automaticamente Obsidian -> Open WebUI.

## Gate futuro
Para API real sera necesaria autorizacion explicita con frase dedicada y evidencia de dry-run PASS.
