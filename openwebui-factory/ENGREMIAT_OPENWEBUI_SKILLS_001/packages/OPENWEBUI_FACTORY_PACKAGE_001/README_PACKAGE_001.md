# OPENWEBUI FACTORY PACKAGE 001

## Objetivo
Primer paquete usable para reducir el proceso manual de creación de Skills y Knowledge en Open WebUI.

## Contenido
- skills/: Skills documentales generadas desde Obsidian.
- knowledge-fast/: documentos mínimos para Knowledge FAST.
- tests/: prompts de validación manual.
- manifest/: manifiesto del paquete.
- manual/: guía de importación asistida.

## Seguridad
api_real_write=False
openwebui_write=False
manual_assisted_import=True
external_actions=False
tools_functions=False

## Orden recomendado
1. Revisar Skills.
2. Importar o crear manualmente 1 Skill de prueba.
3. Importar Knowledge FAST mínimo.
4. Probar prompts de tests.
5. Registrar resultado antes de automatizar API.
