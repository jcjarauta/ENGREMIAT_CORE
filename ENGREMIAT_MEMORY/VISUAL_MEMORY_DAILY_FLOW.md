# VISUAL_MEMORY_DAILY_FLOW

## Rutina diaria
1. Ejecutar `.\eng.ps1 memory-status` para ver estado resumido.
2. Ejecutar `.\eng.ps1 memory-sync` al terminar un bloque de desarrollo.
3. Ejecutar `.\eng.ps1 memory-dashboard` para abrir el panel visual.
4. Revisar gates, errores, readiness y next.
5. Abrir Logseq si se necesita navegación profunda.

## Regla
La memoria visual informa y ordena; no ejecuta acciones sensibles.
