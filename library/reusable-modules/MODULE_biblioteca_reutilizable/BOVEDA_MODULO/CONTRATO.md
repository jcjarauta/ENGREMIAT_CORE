# CONTRATO MODULE_biblioteca_reutilizable

Nombre: Biblioteca reutilizable
Estado: AVAILABLE_FOR_PROJECT_REUSE

## Objetivo
Promover modulos cerrados a biblioteca, indexarlos y reutilizarlos en nuevos proyectos.

## Uso previsto
Este modulo puede copiarse dentro de un proyecto Engremiat nuevo y adaptarse al objetivo concreto.

## Criterio de cierre
El modulo reutilizado debe tener objetivo definido, tareas creadas, smoke PASS y evidencias registradas.
