# ENGREMIAT - Dossier base para partners

Engremiat es una plataforma modular para coordinación, memoria viva, documentación y futura ejecución asistida por workers.

## Qué aporta
- Orden del proyecto y sus módulos.
- Documentación reutilizable.
- Trazabilidad de decisiones y avances.
- Preparación para pilotos colaborativos.
