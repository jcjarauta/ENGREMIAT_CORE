# Convocatoria Demo Folder

Entidad convocante: Agencia Demo Folder de Innovacion Social.
Territorio: Catalunya y España.
Fecha limite de presentacion: 30/11/2026.
Importe maximo por proyecto: 75000 EUR.
Modalidad: subvencion competitiva para proyectos de entrada documental estructurada.

Objeto de la convocatoria: financiar soluciones que conviertan documentos, carpetas y evidencias locales en fichas reutilizables para gestion de proyectos, modulos, tareas y dossiers.

Beneficiarios: entidades sociales, cooperativas, administraciones colaboradoras y comunidades locales.

Criterios de valoracion: impacto social, reutilizacion de datos, trazabilidad, viabilidad tecnica, sostenibilidad y escalabilidad.

Documentacion requerida: memoria tecnica, presupuesto, cronograma, indicadores, declaracion responsable y anexos tecnicos.

Resumen: carpeta demo para validar que Data Intake registra carpetas locales como fuente compuesta.
