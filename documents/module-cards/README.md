# ENGREMIAT_MODULE_CARD_CANON_001

Objetivo: consolidar el formato canonico de tarjetas vinculadas a modulos antes de construir autodiagnostico o workers propositivos.

Decision: la tarjeta es la unidad minima de propuesta, tarea, mejora o bug. El modulo es la unidad de capacidad. El sistema puede proponer tarjetas, pero el humano aprueba antes de ejecutar.
