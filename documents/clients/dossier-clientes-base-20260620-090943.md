# ENGREMIAT - Dossier base para clientes / usuarios

Engremiat ayuda a ordenar un proyecto, convertir ideas en módulos, revisar el estado y mantener documentación viva sin perder el hilo.

## Beneficios
- Menos dispersión.
- Más trazabilidad.
- Mejor continuidad entre sesiones.
- Preparación para automatización asistida.
