# Paleta canonica de consola ENGREMIAT

Fecha: 2026-06-20T10:10:49

## Objetivo
Dar coherencia visual a todo el sistema sin sobrecargar la pantalla.

## Paleta
- Titulo: Cyan
- Ruta: DarkGray
- Rol: Gray
- Principio: DarkGray
- Seccion activa: Yellow
- OK: Green
- WARN: Yellow
- ERR: Red
- NEXT: Cyan
- Texto normal: Gray
- Texto secundario: DarkGray

## Norma
El color tiene significado, no decoracion. La interfaz debe seguir siendo clara aunque no se perciba el color.

## Uso recomendado
Usar helpers comunes: Title, Route, Role, Principle, Section, OK, Warn, Err, Next, Muted.
