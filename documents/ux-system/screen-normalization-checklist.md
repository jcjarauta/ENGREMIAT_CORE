# Checklist para normalizar pantallas

## Antes de tocar
- [ ] Identificar pantalla legacy funcional.
- [ ] Hacer backup.
- [ ] Crear contrato o mapa de pantalla.
- [ ] Definir rol: productor, consumidor, supervisor, exportador o estado.

## Diseño
- [ ] Header con ruta, rol y principio.
- [ ] Acciones agrupadas por conceptos.
- [ ] Enter refresca.
- [ ] b vuelve.
- [ ] ? ayuda.
- [ ] Prompt nombrado por pantalla.

## Cableado
- [ ] Un submenu por fase.
- [ ] Parse PSParser.
- [ ] Report JSON.
- [ ] Prueba humana mínima.
- [ ] No reemplazar acceso principal hasta validar.

## Cierre
- [ ] Estado PASS documentado.
- [ ] Riesgos/gates registrados.
- [ ] Siguiente submenu recomendado.
