# ENGREMIAT - Dossier base para subvenciones

## Problema
Muchos proyectos comunitarios, cooperativos o de economía circular tienen dificultad para coordinar trabajo, documentar avances, justificar impacto y convertir conocimiento disperso en acciones trazables.

## Solución
Engremiat propone una infraestructura local-first para organizar proyectos, módulos, memoria, documentación, workers y participación humana con trazabilidad.

## Innovación
- Memoria viva generada desde evidencias reales.
- Documentación automática desde estado del sistema.
- Preparación para workers locales y exportación documental.
- Enfoque modular y replicable.
