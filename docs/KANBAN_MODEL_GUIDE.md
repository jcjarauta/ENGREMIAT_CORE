# KANBAN MODEL GUIDE

## Proposito
Kanban es una superficie humana de lectura y control. No es la fuente canónica del codigo ni un motor de ejecucion.

## Columnas visuales
- BACKLOG
- READY
- GATED
- RUNNING
- REVIEW
- DEFERRED
- DONE
- BLOCKED

## Regla central
Un solo escritor activo. La tarjeta refleja estado y siguiente accion; no ejecuta por si sola.

## Campos de tarjeta
- objective
- stage
- block
- step
- state
- risk
- required
- priority
- status
- notes
- next_action

## Estados visuales
BACKLOG, READY, GATED, RUNNING, REVIEW, DEFERRED, DONE, BLOCKED

## Seguridad
- Kanban como vista
- Sin apply
- Sin Git write
- Sin remote execution
- Sin shared repo edit
