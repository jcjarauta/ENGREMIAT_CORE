# E05 · Canvas global y de proyecto

## Decisión
Se crean tres mapas Canvas: sistema global, control operativo y proyecto plantilla.

## Uso
- SYSTEM_CANVAS muestra arquitectura ENGREMIAT global.
- CONTROL_HUB.canvas muestra el espacio único humano.
- PROJECT_CANVAS.canvas muestra la secuencia de un proyecto.

## Regla
Canvas visualiza relaciones. No sustituye gates, reports ni evidencias canónicas.
