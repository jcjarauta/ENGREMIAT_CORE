# ENGREMIAT · Canvas oficial agrupado

## Resultado
Canvas global agrupado creado para consolidar la arquitectura oficial.

## Idea central
El humano decide si crea bóveda maestra, módulo relacionado, project space o superficie de trabajo. ENGREMIAT ubica, agrupa, enlaza y deja visible.

## Regla
Todo nuevo módulo debe declararse dentro de un grupo oficial y enlazarse con su contexto, gates, evidencias y siguiente acción.
