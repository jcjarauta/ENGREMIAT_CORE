# KANBAN REAL SETUP GUIDE

## Proposito
Dashboard local humano para ver el estado del Kanban real.

## Flujo por PC
- MAIN-PC: revisar tablero, validar estado, preparar continuidad.
- old-pc: leer tablero, preparar handoff y seguir sin escribir en paralelo.
- Un solo escritor activo por objetivo o bloque.

## Reglas
- Kanban es una vista.
- No apply.
- No Git write.
- No remote execution.
- No shared repo edit.
