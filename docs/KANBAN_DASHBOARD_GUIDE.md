# KANBAN DASHBOARD GUIDE

## Proposito
Dashboard local humano para ver el estado del Kanban real.

## Uso
1. Leer el resumen superior.
2. Revisar conteos por columna.
3. Abrir la tabla de tarjetas y seguir la siguiente accion.

## Reglas
- Kanban es una vista.
- No apply.
- No Git write.
- No remote execution.
- No shared repo edit.
