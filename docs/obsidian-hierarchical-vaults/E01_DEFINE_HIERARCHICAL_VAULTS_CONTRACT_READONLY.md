# ENGREMIAT - E01 contrato jerarquico de bovedas

## Objetivo
Separar vision global, vision de proyecto y vision de modulo para reducir ruido visual en Obsidian.

## Jerarquia
- L0 MASTER: mapa global y control.
- L1 PROJECT: espacio visual del proyecto.
- L2 MODULE_SPACE: detalle del modulo dentro del proyecto.
- L3 MODULE_VAULT opcional: solo para modulos grandes.

## Reglas clave
- La master no debe contener contenido profundo.
- El proyecto contiene roadmap y modulos del proyecto.
- El modulo contiene contrato, estado, evidencia, canvas y capas.
- Los artefactos tecnicos visibles necesitan nota humana puente.
- Los huerfanos se integran, clasifican u ocultan.
