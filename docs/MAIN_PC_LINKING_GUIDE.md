# MAIN PC LINKING GUIDE

## Objetivo
Ver y operar el vínculo real con MAIN-PC sin mezclar responsabilidades.

## Reglas
- Git es la fuente de verdad.
- ENGREMIAT_SHARED es puente, no repo vivo.
- Un solo writer activo por objetivo o bloque.
- No browser auto por defecto.

## Flujo
1. Revisar dashboard.
2. Leer handoff.
3. Validar repo local.
4. Mantener sync seguro.
5. Cerrar con manifest y close report.

## Prohibiciones
- No secretos
- No .env
- No tokens
- No git write automatico
- No auto apply
- No shared repo edit
