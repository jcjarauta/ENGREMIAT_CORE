# HUMAN_COMMANDS

Comando unico: `.\eng.ps1 <comando>`

Comandos principales:
- `start` abre la pantalla inicial humana.
- `status` muestra estado operativo.
- `next` muestra el siguiente paso.
- `readiness` muestra el cierre/readiness de E37.
- `gates` muestra la politica de gates humanos.
- `artifacts` lista evidencias locales.
- `context` muestra el mapa del Command Hub.
- `errors` lista reportes de error.
- `docs` muestra este manual.
- `dashboard` / `mvp-open` abre el panel humano.
- `prompt` prepara prompt de worker sin enviarlo.
- `inbox` muestra bandeja local de evidencias/propuestas.
- `review` revisa propuesta sin aplicar.
- `approve` solo declara gate pendiente; no aplica cambios.

Seguridad: apply_executed=false, auto_apply=false, remote_execution=false, git_write=false.
