# TOOL CALIBRATION CONTROLLER

## Objetivo

Automatizar la calibracion asistida de Knowledge/bundles/prompts sin llamar automaticamente a IA.

## Comandos

```powershell
.\tools\tool-calibration-controller\Invoke-ToolCalibrationController.ps1 -Mode status
.\tools\tool-calibration-controller\Invoke-ToolCalibrationController.ps1 -Mode prepare
.\tools\tool-calibration-controller\Invoke-ToolCalibrationController.ps1 -Mode intake -ResponseFile "ruta\a\respuesta.md"
```
