# 02 GUARDS DE AMPLIACION

## Guard obligatorio
- Validar baseline antes de tocar.
- Crear script separado.
- Validar parse.
- Generar reporte JSON.
- Validar baseline despues.

## No permitido
- No reescribir launcher entero para una extension.
- No meter logica compleja en router.
- No borrar ni sustituir plantilla maestra oficial.
- No crear plantillas inventadas.
- No romper comandos GLOBAL/PROYECTO/MODULO.
- No ocultar errores.

## Si un cambio toca contrato
Crear freeze successor: operator-baseline-freeze.v2.json.
