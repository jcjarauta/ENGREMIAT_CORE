# 01 NORMAS DE USO Y UX

## Normas permanentes
- Enter vacio, refrescar o f5 redibuja la pantalla.
- ? muestra ayuda contextual.
- comandos o menu muestra el menu actual.
- b, a, atras, volver o 0 vuelve al nivel anterior cuando aplique.
- Toda pantalla debe mostrar UBICACION: Nivel, Proyecto, Modulo y Ruta.
- El usuario debe saber siempre donde esta situado.
- El operador debe evitar pantallas sin salida.
- Los errores deben quedarse visibles y guardarse en reports.

## Normas de interaccion
- No pedir confirmaciones innecesarias si el siguiente paso logico esta claro.
- Si hay error, capturar evidencia y detener.
- No ocultar errores volviendo automaticamente al menu.
- Cada nuevo flujo debe tener retorno seguro.
