# MANUAL HUMANO-MAQUINA - ENGREMIAT_MODULE_INTAKE_REGISTRY_001

## Uso
Normaliza y ordena la entrada de nuevos modulos en ENGREMIAT.

## Comandos de etapa
- E01_INTAKE_FOUNDATION
- E02_INTAKE_VALIDATOR
- E03_CANONICAL_SCAFFOLDER
- E04_REGISTRY_AND_DASHBOARD
- E05_CLOSE_AND_NEXT

## Archivos clave
- module.json
- contracts/module-intake-contract.json
- schemas/module-intake-schema.json
- packages/module-intake-package.json
- registry/module-intake-registry.json
- E02_INTAKE_VALIDATOR/packages/module-intake-validator.json
- E03_CANONICAL_SCAFFOLDER/contracts/canonical-scaffold-contract.json
- E04_REGISTRY_AND_DASHBOARD/dashboard/index.html
- E05_CLOSE_AND_NEXT/contracts/module-intake-close-manifest.json

## Regla
Antes de crear cualquier modulo nuevo, definir module.json, validar schema, crear scaffold y registrar decision.
